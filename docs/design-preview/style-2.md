# 风格方案二：赛博极光风

## 一句话定义
**赛博极光风：暗色宇宙中的代码诗人美学**

---

## 设计理念与情绪参考

这个风格将技术博客的"身份认同"最大化——**让博客本身就像一个精心设计的 IDE 主题**。受到黑客文化、Terminal 美学和北极光色谱的双重影响，在深邃的近黑色背景上用霓虹渐变构建视觉焦点。不是粗糙的"赛博朋克"刻板印象，而是在极暗背景上用克制的发光色彩传递技术精英的审美自信。

情绪参考词：深邃、技术感、自信、专注、夜晚编程、宇宙感

---

## 色彩体系（60-30-10 法则）

| 角色 | 色值 | 说明 |
|------|------|------|
| 60% 主背景 | `#0E0E14` | 深蓝黑，比纯黑多一点宇宙感 |
| 30% 辅助色 | `#1A1A2E` / `#16213E` | 深蓝紫色系，用于卡片背景和分区 |
| 10% 强调色 | 渐变 `#A855F7` → `#06B6D4` | 紫→青渐变，用于交互元素、边框光效、重点标签 |

**深色主题（主要主题）配色补充**：
- 主文字：`#E2E8F0`（冷白，与暖背景形成色温对比）
- 辅助文字：`#94A3B8`（蓝灰，用于日期、元信息）
- 代码高亮：`#10B981`（翠绿，Terminal 经典色）
- 错误/警告：`#F87171`（柔红）
- 卡片边框：`rgba(168, 85, 247, 0.2)` 低透明度紫色描边
- 卡片 hover 发光：`box-shadow: 0 0 20px rgba(168, 85, 247, 0.3)`

**浅色主题**（此风格主推深色，浅色作为可选）：
- 背景：`#F8FAFC`（极浅蓝灰）
- 强调色变暗：`#7C3AED` / `#0891B2`
- 文字：`#1E293B`

## 字体方案

### 标题字体：JetBrains Mono（代码等宽字体）
- 选择理由：在个人技术博客中，用代码字体做标题是反常规的惊喜之举，直接声明"我是开发者，这里是我的领土"。等宽字体的机械美感在暗色背景上极具辨识度。
- 使用场景：页面大标题、博客名称
- 字重：800（ExtraBold）

### 正文字体：思源黑体（Noto Sans SC）
- 选择理由：无衬线黑体与等宽标题形成"机械 + 清晰"的统一调性，高可读性适合长文阅读，在暗色主题下白色黑体比宋体更清晰。
- 使用场景：正文、导航、UI 文字
- 字重：300（Light）—— 深色背景下细字重更优雅

### 字体层级定义

| 层级 | 字号 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| 大标题 H1 | 52px (桌面) / 34px (移动) | 800 JetBrains | 1.1 | -0.03em | 页面 Hero 标题 |
| 小标题 H2 | 22px / 18px | 500 JetBrains | 1.3 | -0.01em | 文章卡片标题 |
| 正文 Body | 15px / 14px | 300 黑体 | 1.85 | 0.01em | 文章内容 |
| 辅助说明 Caption | 12px / 11px | 400 黑体 | 1.5 | 0.05em | 日期、标签（全大写字母部分用 letter-spacing 拉宽）|

## 布局风格描述

**整体策略：暗色宇宙中的网格构建**

- 桌面端：非居中的偏左布局（内容区 65% + 右侧边栏 30%），最大宽度 1100px
- 文章列表：卡片网格（2列），卡片有微妙的紫色描边，hover 时描边发光
- 导航：毛玻璃顶部导航（`backdrop-filter: blur(12px)`），滚动后边框微发光
- 左侧博客名称用 JetBrains Mono 等宽字体 + 光标闪烁动效（Terminal 打字机效果）
- 背景装饰：极淡的几何网格线（`opacity: 0.03`），不喧宾夺主但增加层次感
- 右侧边栏：个人简介快照 + 标签云（标签用渐变描边样式）

---

## 核心页面氛围描述

**首页（文章列表）**：打开是一片深邃的近黑宇宙，博客名称用等宽字体打字机效果逐字出现（_光标在最后闪烁_）。文章卡片以 2 列网格排布，每张卡片有淡紫色边框，悬停时边框发出柔和的渐变光晕——就像在黑暗中触碰到一块发光的水晶。标签用紫→青渐变背景色渲染，极具辨识度。

**文章详情页**：沉浸式阅读。右侧浮动目录（TOC）用紫色高亮当前阅读章节。代码块有终端样式（顶部三个颜色圆点 + 等宽字体 + 深绿代码色），复制按钮悬停时出现光效。行内代码用 `background: rgba(168, 85, 247, 0.15)` 紫色淡背景包裹。

---

## 适合的场景和目标人群

- 后端/全栈开发者博主，以技术身份认同为核心
- 习惯深色 IDE 主题（VS Code Dark+、Dracula）的开发者
- 希望博客本身就展示技术审美的创作者
- 与 GitHub 黑色、Vercel 黑白风格形成呼应但又更有个性
